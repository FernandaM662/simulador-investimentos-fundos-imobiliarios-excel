# Simulador de Investimentos em Fundos Imobiliários (Excel)

##  Descrição do Projeto
Este projeto consiste no desenvolvimento de uma planilha em Excel para simulação de investimentos em fundos imobiliários (FIIs).  
O objetivo é auxiliar investidores a visualizarem o valor total investido, o patrimônio acumulado e os dividendos mensais ao longo do tempo.

Projeto desenvolvido como parte do desafio prático da plataforma **DIO**.

---

##  Objetivos de Aprendizagem
- Criar ferramentas de simulação de investimentos no Excel  
- Aplicar cálculos financeiros, como rendimento mensal e dividendos  
- Automatizar cálculos para apoio à tomada de decisão  
- Documentar processos técnicos de forma clara  
- Utilizar o GitHub para versionamento e documentação  

---

##  Funcionalidades da Planilha
- Simulação de valor investido ao longo do tempo  
- Cálculo de patrimônio acumulado  
- Estimativa de dividendos mensais  
- Interface simples e intuitiva para o usuário  

---

##  Ferramentas Utilizadas
- Microsoft Excel  
- GitHub
